# here-me-now

**Patient Self Check-in via NOSTR**

Ein dezentraler, datenschutzfreundlicher Check-in für Arztpraxen – basierend auf dem NOSTR-Protokoll.

## Konzept

- **Migrierbarer Identifier**: Patienten nutzen ihre nsec/npub als digitale Identität
- **Keine zentrale Datenbank**: Daten werden auf NOSTR-Relay-Netzwerk verteilt
- **Sofortiger Check-in**: Patienten melden sich selbstständig über Web-App oder Smartphone-App
- **Praxis-Interface**: Ärzte empfangen Check-ins in Echtzeit

## Technologie
- NOSTR Protocol (NIP-01, NIP-04 Encryption)
- Web App (React/Vue) + optional native Apps
- Lightning-Integration für optionale Zahlungen
